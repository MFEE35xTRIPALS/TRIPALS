# TRIPALS 專案說明

### npm install

因為使用 create-react-app 建立專案時有許多的相依套件在 node_modules 資料夾內，.gitignore 預設不會將 node_modules 一同推送，從 GitHub Clone 下來後，需要在 cmd 內輸入 npm install 抓取 package 內的套件。
